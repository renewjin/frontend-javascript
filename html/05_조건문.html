조건문

            프로그래밍 코드가 해석되는 순서를 제어하는 제어문의 한 종류로
            작성된 조건식에 따라 수행되는 코드를 변경하는 구문

if 문

            
                if (조건식){
                    조건식이 true일 경우 실행할 코드;
                }
            
            * 조건식 : 결과가 true, false인 식

if-else 문

            
                if(조건식) {
                    조건식이 true인 경우 수행할 코드;
                }
            
            *else는 조건식을 작성하는 () 없음
0~100 사이에 랜덤숫자를 발생시켜서 홀짝 판별

if() - else if() - else()

            
                if(조건식1) {
                    조건식1 이 true인 경우 실행
                } else if (조건식2) {
                    조건식1 false 조건식2 이 true인 경우 실행
                } else {
                    조건식1, 2 모두 false 경우 실행
                }
어린이 청소년 성인 구별
            나이를 입력 받아 0 ~ 13 : 어린이
            14 ~ 19 : 청소년
            20 ~ : 성인
